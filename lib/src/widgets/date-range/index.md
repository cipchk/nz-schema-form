默认小部件，一般用于日期选择。

## Schema

参数 | 说明 | 类型 | 默认值
----|------|-----|------
placeholder | 在文字框中显示提示讯息  | `string` | -
readOnly | 是否只读状态  | `boolean` | -
widget | 组件  | `string | object` | - 


## Widget Params

参数 | 说明 | 类型 | 默认值
----|------|-----|------
id | 组件名称  | `string` | `date`
format | 日期格式  | `string` | - 
showTime | 是否显示时间选择器 | `boolean` | `false`
start |  开始placeholder | `string`| `start`
end |  结束placeholder | `string`| `end`

## Demo

**各种格式的配置**

```json

{
	"dateRange": {
		"type": "string",
		"title": "日期范围",
		"widget": {
			"id": "date-range"
		}
	}
}
```
